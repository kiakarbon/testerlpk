Aplikasi web berbasis Streamlit untuk mencatat hasil praktik dan mengkalkulasi hasil PSA (Particle Size Analysis) nanomaterial. Mendukung ekspor ke format Word dan PDF.

### ğŸ“ Sistem Pencatatan Praktik
- Form input lengkap untuk eksperimen nanomaterial
- Upload gambar hasil sintesis (maks 5 gambar)
- Kategorisasi berdasarkan jenis nanomaterial dan metode sintesis
- Penyimpanan data lokal dengan session management
- Ekspor ke format Word (.docx) dengan template profesional

### ğŸ§® Kalkulator PSA
- Input data distribusi ukuran (Diameter, % Volume, PDI)
- Upload data dari file Excel/CSV
- Analisis statistik otomatis (mean, median, std dev, variance)
- Perhitungan PDI (Polydispersity Index)
- Klasifikasi kualitas nanomaterial berdasarkan PDI
- Visualisasi dengan Plotly (distribusi, histogram, scatter plot)
- Ekspor hasil ke PDF dengan grafik dan statistik lengkap

### ğŸ“Š Manajemen Data
- Penyimpanan data dalam session
- Filter dan pencarian data
- Edit dan hapus data
- Backup data otomatis
- Ekspor batch untuk multiple data

### ğŸ“ Sistem Ekspor
- **Word Export**: Template profesional untuk laporan praktikum
- **PDF Export**: Laporan PSA dengan grafik dan statistik
- **Excel Export**: Data mentah dalam format spreadsheet
- **Batch Export**: Ekspor multiple data sekaligus
